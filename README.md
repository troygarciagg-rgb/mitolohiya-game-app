# Mitolohiya: Ang Kasunduan

Static landing page for **Mitolohiya: Ang Kasunduan**.

## Run locally

Open `index.html` in your browser.

## Deploy

This project is designed to deploy on Vercel as a static site.

## APK download

The Android APK should be distributed via **GitHub Releases** (not committed to git, due to GitHub file-size limits).
